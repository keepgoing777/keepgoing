변수와 연산자

1. 변수: 값을 담는 공간 / 변수명 : 값

  • "var" 재할당(변경가능) 구버전이라 잘 안씀
  • "let" 자유선언, 변경값이 많을 때 사용
  • "const" 상수선언, 재할당 불가능, 개발 실무에서 많이 씀

2. 자료형과 typeof

console.log(변수) : string, number, boolean, undefined, null 등

3. 문자열과 템플릿 리터럴

  • "할말" + 변수 + "할말"
  • `할말 ${변수} 할말`
  • "할말 \n (문단나누기), \t(탭) 할말"

4. 숫자

사칙연산 동일

  • num1**num2 = num1 제곱을 num2만큼 / num1 % num2 = num1 나누기 num2 하고 나머지 값
  • 문자>숫자 : console.log(Number("100")) "100"문자를 Number로
  • 숫자>문자 : console.log(String(300)) 300 숫자를 String으로
  • 소수제거 정수화 : parseInt(3.9)
  • 문자열>실수화 : parseFloat("4.2")

5. Boolean과 비교연산자

== 모양이 같냐 / === 타입까지 같냐 / Boolean("") false, Boolean("0") true

6. 논리연산자

&& 둘다 true일때 true, || 둘중 하나만 true여도 true, ! 부정, !!이중부정

조건문

1.if-else 조건문 : if (조건) {조건이 true인 경우} else{조건이 false인 경우}

2. else-if 조건문 : if (조건A) {조건 A가 true인 경우} else if (조건B) {조건 A가 false면서 조건 B가 true인 경우}else{모든 조건이 false}

삼항연산자 구조: 조건 ? a (true) : b (false)
다항연산자 구조: 조건 a? 1 : 2 ; 조건 b? 3 : 4 ; 조건 c? 5 : 6 ;

3. switch 조건문 : switch(데이터) {case 조건 A : 조건 A가 true break; case 조건 B : 조건 B가 true면 break; default : 모든 조건 false}

4. 중첩 if문 : if(id==="user1")if(pw==="1234"){console.log("로그인성공");}else{console.log("비밀번호 오류");}{else{console.log("아이디 오류");}}

5. break; 가 없는 switch : 조건에 해당하는 문구 그 이후의 결과값도 모두 출력

배열 : 하나의 변수에 여러값을 저장하는 자료형

  • const arr=[1,2,3,4,5]
  • console.log(JSON.Stringify(arr)): 배열 바로 문자열로 보이게함
  • console.log(typeof arr): 타입이 뭔지 (object)
  • console.log(ArrayisArray(arr)) 배열이 맞는지
  • console.log(arr.length) 해당 배열의 길이(많이 씀)
  • console.log(arr[arr.length-1]) 마지막 배열 값
  1. 배열 추가: 맨 뒤 arr.push(n) 맨 앞 arr.unshift(n) arr.slice(a,b,c) 인덱스 a번째, 삭제할개수b개, 추가할값 c
  2. 배열 삭제: 맨 뒤 arr.pop(n) 맨 앞 arr.shift(n) arr.slice(a,b) 인덱스 a번째, 삭제할개수 b개
  3. 배열값 수정: arr.slice(a,b,c) or arr.indexof(n) n의 인덱스 찾기 사용
  4. 배열 정렬: arr.sort(a,b=> a-b); 오름차순 arr.sort(a,b=>b-a); 내림차순 arr.reverse(); 반대로
  5. 문자에서 배열: const strToArr = str.split(",");
    배열에서 문자로: const arrToStr = strToArr.join(",");
  6. 빈 배열에 값 추가: const arr2 = []; arr2.push(1); arr2unshift(5)
  7. slice 자르기: slice(n,m) n에서부터 m까지(m 미포함)
  8. 구조 분해 할당: 여러값 동시에 여러 변수에 할당
  9. 중복제거: const unique = newSet(arr3) 그후 배열화
